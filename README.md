# CoreGameSample

Simple multiplayer game built with .NET CORE, SignalR, PhaserJS, webpack.

# How to run

1. Clone the repository

then

- `dotnet restore`
- `npm install`
- `typings install`
- `webpack`
- `dotnet run`

Or...

- `dotnet restore`
- `dotnet publish SOME_PATH`
- `dotnet SOME_PATH/Game.dll`
